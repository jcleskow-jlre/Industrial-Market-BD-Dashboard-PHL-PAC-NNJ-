# PA-NJ Industrial Market & BD Dashboard - GitHub upload package

This package is structured for GitHub Pages. Upload the CONTENTS of this folder to the root of the repository, not the folder itself.

Required root files/folders:
- index.html
- dashboard.html
- data/
- data_qa/
- import_templates/
- DATA_IMPORT_QA.json
- README_IMPORT.md
- README_GITHUB_UPLOAD.md
- .nojekyll

GitHub Pages settings:
1. Go to the repository.
2. Click Settings.
3. Click Pages.
4. Under Build and deployment, set Source to Deploy from a branch.
5. Select the main branch and /root.
6. Save.

Launch path:
- index.html is the landing page.
- dashboard.html is the main dashboard.
