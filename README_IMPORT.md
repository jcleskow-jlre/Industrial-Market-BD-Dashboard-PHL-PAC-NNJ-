# PHI-PAC-NNJ Industrial Market & BD Dashboard - GitHub upload package

Upload the contents of this ZIP to the root of the GitHub repository. Do not upload the ZIP file itself.

Required root files/folders:
- index.html
- dashboard.html
- data/
- data_qa/
- import_templates/
- .nojekyll

GitHub Pages settings:
- Source: Deploy from a branch
- Branch: main
- Folder: /root

After uploading, wait 1-3 minutes for GitHub Pages to redeploy, then hard-refresh the browser.
